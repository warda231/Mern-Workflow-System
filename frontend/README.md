# MERN Workflow Approval System

A workflow approval system inspired by enterprise business processes.

## Features

- JWT Authentication
- Role-based authorization
- Workflow approval system
- Dashboard metrics
- Search, filtering, pagination
- Responsive UI

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT
